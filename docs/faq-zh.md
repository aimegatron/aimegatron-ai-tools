# ❓ 常见问题

## 🤖 一般问题

### 什么是AIMEGATRON？
AIMEGATRON是一套Chrome扩展程序，可在4个主要AI平台上自动化批量问题处理：ChatGPT、Gemini、Grok和Perplexity。核心功能完全免费。

### 真的免费吗？
是的！AIMEGATRON的核心功能完全免费，没有隐藏费用或使用限制。

### 支持哪些AI平台？
- 🤖 **ChatGPT** (chatgpt.com)
- 🔮 **Gemini** (gemini.google.com)
- ⚡ **Grok** (grok.com)
- 🔍 **Perplexity** (perplexity.ai)

### 我需要这些AI平台的账户吗？
是的，您需要在想要使用的AI平台上拥有活跃账户。大多数平台都提供免费版本。

## 🔧 安装和设置

### 如何安装扩展？
请查看我们的[安装指南](installation-zh.md)获取详细说明。您可以从Chrome网上应用店安装或直接下载。

可以！您可以同时安装和使用所有四个工具。

## 🛠️ 故障排除

### 扩展图标没有出现
1. 检查 `chrome://extensions/` 确保已安装并启用
2. 尝试将其固定到工具栏
3. 刷新AI平台页面

### 问题没有处理
1. 确保您已登录AI平台
2. 检查网络连接
3. 尝试在设置中降低处理速度
4. 确保AI平台没有停机

### 导出不工作
1. 检查浏览器是否允许下载
2. 尝试不同的导出格式（JSON vs CSV）
3. 确保您对下载文件夹有写权限

### 扩展运行缓慢
1. 关闭其他浏览器标签以释放内存
2. 减少批处理中的问题数量
3. 在设置中增加问题间的延迟

## 💡 技巧和最佳实践

### 如何获得更好的结果？
1. **问题要具体**
2. **对类似问题使用一致的格式**
3. **先用小批量测试**
4. **在可用时启用网络搜索** 获取更新信息

### 如何组织大批量？
1. **将类似问题分组** 在一起
2. **导出时使用描述性文件名**
3. **分小块处理** 便于管理
4. **保留问题列表备份**

### 如何最大化生产力？
1. **提前在文本文件中准备问题**
2. **对重复问题类型使用模板**
3. **设置适当延迟** 避免速率限制
4. **定期导出** 避免数据丢失
