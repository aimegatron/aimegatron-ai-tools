## ğŸ› ï¸ Troubleshooting

### The extension icon doesn't appear
1. Check `chrome://extensions/` to ensure it's installed and enabled
2. Try pinning it to the toolbar
3. Refresh the AI platform page

### Questions aren't processing
1. Make sure you're logged into the AI platform
2. Check your internet connection
3. Try reducing the processing speed in settings
4. Ensure the AI platform isn't experiencing downtime

### Export isn't working
1. Check if your browser allows downloads
2. Try a different export format (JSON vs CSV)
3. Ensure you have write permissions to the download folder

### Extension seems slow
1. Close other browser tabs to free up memory
2. Reduce the number of questions in your batch
3. Increase the delay between questions in settings
