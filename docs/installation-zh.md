# 📥 安装指南

本指南将帮助您安装任何AIMEGATRON AI生产力工具。

## 🚀 快速安装（推荐）

### 方式1：Chrome网上应用店
安装任何AIMEGATRON工具的最简单方法：

1. **选择您的工具**：
   - [🤖 ChatGPT工具](https://chromewebstore.google.com/detail/chatgpt-bulk-auto-qa-gene/bbfbgifbgbjidbnopolnnhkgakejhgfd)
   - [🔮 Gemini工具](https://chromewebstore.google.com/detail/ndafnldalboekcbgcpbikliagghcjlpo)
   - [⚡ Grok工具](https://chromewebstore.google.com/detail/grok-bulk-auto-qa-generat/olmfakojjcdbdaabmoaphpgiimpbnjah)
   - [🔍 Perplexity工具](https://chromewebstore.google.com/detail/perplexity-bulk-auto-qa-g/ccmjoonieapkljbechafcnnlcdkfanbc)

2. **点击"添加至Chrome"**
3. **确认安装** 当提示时
4. **固定扩展** 到工具栏以便访问

## 🔧 手动安装

### 方式2：从GitHub直接下载
如果您喜欢手动安装：

1. **下载扩展**：
   - [ChatGPT扩展ZIP](../downloads/extension-chatgpt.zip)
   - [Gemini扩展ZIP](../downloads/extension-gemini.zip)
   - [Grok扩展ZIP](../downloads/extension-grok.zip)
   - [Perplexity扩展ZIP](../downloads/extension-pplx.zip)

2. **解压ZIP文件** 到计算机上的文件夹

3. **打开Chrome扩展页面**：
   - 在地址栏输入 `chrome://extensions/`
   - 或转到菜单 → 更多工具 → 扩展程序

4. **启用开发者模式**：
   - 切换右上角的"开发者模式"开关

5. **加载扩展**：
   - 点击"加载已解压的扩展程序"
   - 选择您解压ZIP文件的文件夹

6. **固定扩展**：
   - 点击工具栏中的拼图图标
   - 找到您的AIMEGATRON工具并点击固定图标

## ✅ 验证安装

安装后，您应该看到：

- ✅ 扩展列在 `chrome://extensions/` 中

## 🛠️ 故障排除

### 扩展没有显示？
1. 检查是否已安装：`chrome://extensions/`
2. 确保已启用（切换开关打开）
3. 尝试将其固定到工具栏
4. 刷新AI平台页面

### 安装失败？
1. 确保您有Chrome 88+或兼容浏览器
2. 尝试重新下载（文件可能已损坏）
3. 暂时禁用其他扩展
4. 联系支持：aimegatronweb@gmail.com
